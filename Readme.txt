처음 실행시 주 화면에는 아무런 과제도 등록되어 있지 않습니다.

"추가" 버튼을 눌러 과제를 추가할 수 있습니다.
단순히 언제까지 해야 하는 과제면 시작일과 마감일을 동일하게 적용하면 됩니다.
추가 버튼을 눌렀을 떄 제공되는 입력창을 통해 URL과 FILE을 추가할 수 있습니다.
파일 추가하기의 경우 자신 컴퓨터의 디렉토리에 따라 달라질 수 있음으로 다른 컴퓨터에서는 등록한 파일이 열리지 않을 수 있습니다.

위쪽 상단의 메뉴 스트립에서 "저장"과 "열기" 기능을 사용할 수 있습니다.
초기에 "열기" 기능을 사용하여 이미 과제가 추가되어있는 songwangjin 파일을 열면 
과제를 추가해서 검사해야하는 수고를 덜 수 있습니다.

과제에 변경사항이 생겼을 경우 해당 과제를 클릭하고 우측 하단에서 "수정" 하기 버튼을 눌러 수정할 수 있습니다.
"삭제"를 할 경우 다른 과제를 선택할 때 까지 우측 과제 정보는 남아 있습니다.
해당 과제에 적용된 URL과 파일을 삭제하고 싶을 경우 오른쪽 클릭을 사용하면 됩니다.

이외에도 "필터링 적용하기" 기능을 사용할 수 있으며, 메뉴 스트립을 제외하고는 
화면에서 제공하는 인터페이스 이외의 다른 숨겨진 기능은 없습니다.

*확인 방법*

1.왼쪽 하단의 추가하기 버튼을 눌러 이름,태그,시작일,마감일, 주석,등을 정하고
필요할 경우 URL이나 FILE을 추가한다. 
FILE 추가시 절대 경로를 사용해야 하며 URL경우 http:// 와 같이 전부 적어줘야합니다.
밑에 텍스트 박스가 실제 경로고 위에 텍스트 박스는 단순한 이름입니다.

-> 해당 과정 실행시 파넬에 자신이 입력한 과제가 보입니다.

2.해당 과제를 클릭합니다. ->오른쪽 화면에 정보가 뜨게 됩니다.

3.등록한 URL이나 파일을 눌러 봅시다.
-> 에러 메세지가 뜰 경우 입력을 잘못한 경우 입니다. 
해당 버튼을 오른쪽 클릭해서 지우도록합니다.

4.생각해보니 마감일이 틀렸다 가정하고 수정 버튼을 눌러 마감일을 수정하도록 합시다.

5.우측상단 메뉴 스트립에서 열기 기능을 사용합니다. songwangjin이라는 파일을 열도록 합시다.

6. 저장 되었던 과제리스트가 올라 옵니다. 추가 버튼을 눌러 추가해도 되고 수정해도 됩니다.

7. 필터링 기능을 이용하기 위해 설정후 적용합니다.

8. 메뉴 스트립 "저장"을 이용해 저장하도록 합시다.

9.끝입니다.
